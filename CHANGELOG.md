# Changelog

All notable changes to Zeus are recorded here.

This project follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
in format. It does not follow semantic versioning — Zeus is a
continuously-deployed live trading engine, not a released library.
Changes ship as operational fixes directly to the main branch; there are
no public release tags.

The authoritative change history is in `git log`. This file records
significant operational milestones and structural shifts that are not
obvious from individual commit messages.
