- Mechanism count: 16 distinct mechanisms across 16 files (1:1 in this batch — hit density was low enough that each file resolved to one dominant mechanism except position_truth.py and semantic_types.py which each carry 2-3 related sub-items grouped as one row).
- Buckets: B1 = 7, B2 = 4 (+1 tentative), B3 = 0 standalone (1 B3 finding folds into a B1 row — `CHAIN_QUARANTINED` event type, confirms TAIL-3's dead-code finding), B4 = 8.

This file's `needs_operator_attention()` already groups QUARANTINED with DEGRADED_PROJECTION as a page-a-human bucket — the correct B2 target shape. A migration that only updates `PositionPhase` (the most obvious target, since `LifecyclePhase = PositionPhase`) would silently miss the other two.
