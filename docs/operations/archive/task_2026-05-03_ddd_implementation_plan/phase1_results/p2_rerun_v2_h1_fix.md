# Phase 1 — H1 Denominator Fix Rerun (HIGH metric only)

Created: 2026-05-03
Authority: operator adversarial review (review2.md §3.1, §13)

## What was fixed

The original §2.1/§2.3/§2.4 scripts called `per_day_coverage()` which
returns only dates WHERE observations exist (GROUP BY target_date).
Days with zero observations are absent from the result dict.
Caller code then does `cov_full.get(d_iso)` → `None` → `continue`,
silently excluding zero-coverage days from all downstream analysis.

**H1 fix**: enumerate every calendar date; zero-row days → coverage 0.0.
